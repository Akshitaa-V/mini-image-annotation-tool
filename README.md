# Mini Image Annotation Tool

A lightweight tool for annotating images to support computer vision dataset preparation.

## What it does
- Allows manual labeling/annotation of objects within images
- Outputs structured annotation data (e.g., bounding boxes, labels) for use in training CV models
- Designed to be simple and dependency-light for quick dataset prep tasks

## Tech stack
Python

## Usage
```bash
python annotate.py --input path/to/images/
```

## Notes
Built to support dataset preparation for other computer vision projects in this portfolio.
